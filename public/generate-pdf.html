üìã Quick Instructions

Choose one of the methods below to generate PDF versions of your diagrams:

Method 1: Browser Print to PDF RECOMMENDED

Best for: High quality, all diagrams, works everywhere

  1. Click the button below to open the diagrams page
  2. Press Ctrl+P (Windows) or Cmd+P (Mac)
  3. Select "Save as PDF" as the destination
  4. Choose "Landscape" orientation
  5. Enable "Background graphics"
  6. Click "Save" and choose your location

Method 2: Save Individual Diagrams

Best for: Specific diagrams, presentations

  1. Open the diagrams page using the button above
  2. Click on the diagram tab you want to save
  3. Right-click on the diagram
  4. Select "Save image as..." or "Copy image"
  5. Repeat for each diagram you need

Method 3: Screenshot Tool

Best for: Quick captures, specific sections

  1. Open the diagrams page
  2. Use Windows Snipping Tool or Mac Screenshot (Cmd+Shift+4)
  3. Capture each diagram section
  4. Use an online tool to combine images into PDF

Method 4: Automated Script ADVANCED

Best for: Developers, automation, batch processing

  1. Install Puppeteer: npm install puppeteer
  2. Run: npm run generate:diagrams-pdf
  3. Find PDF in: public/Envirocare-EMS-Diagrams.pdf

Note: Requires Node.js and may take 2-3 minutes to install Puppeteer.

üí° Tips for Best Results

  • Orientation: Use Landscape for better diagram visibility
  • Quality: Enable "Background graphics" in print settings
  • Size: A4 or Letter size works best
  • Margins: Use minimal margins for more diagram space
  • Multiple Pages: Each diagram section will be on a separate page
